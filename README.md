# Лабораторная работа №2
## Построение СКНФ и СДНФ на основании таблиц истинности

### Описание
Программа для анализа булевых функций с выполнением всех требуемых операций лабораторной работы №2.

### Структура проекта

| Файл | Описание |
|------|----------|
| `main.py` | Главный модуль, анализатор булевых функций |
| `parser.py` | Парсинг логических выражений |
| `truth_table.py` | Построение таблиц истинности |
| `normal_forms.py` | Построение СДНФ и СКНФ |
| `zhegalkin.py` | Полином Жегалкина |
| `post_classes.py` | Классы Поста |
| `minimization.py` | Минимизация ДНФ/КНФ |
| `dummy_variables.py` | Фиктивные переменные |
| `boolean_derivatives.py` | Булевы производные |
| `test_laba2.py` | Unit тесты |

### Возможности

1. **Парсинг выражений**: &, |, !, ->, ~ (до 5 переменных a-e)
2. **Таблица истинности**: автоматическое построение
3. **Нормальные формы**: СДНФ, СКНФ (числовая и индексная формы)
4. **Классы Поста**: T0, T1, S, M, L
5. **Полином Жегалкина**: алгебраическая нормальная форма
6. **Фиктивные переменные**: определение и поиск
7. **Булевы производные**: частные и смешанные (порядок 2-4)
8. **Минимизация**:
   - Расчетный метод (с этапами склеивания)
   - Расчетно-табличный метод (с таблицей покрытия)
   - Карта Карно (для 2-4 переменных)

### Запуск

```bash
python3 main.py
```

### Примеры выражений

```
a&b                    # Конъюнкция
!a|b                   # Дизъюнкция с отрицанием
!(!a->!b)|c           # Импликация
a~b                    # Исключающее ИЛИ
(a|b)&!a&!c           # Сложное выражение (вариант 2)
```
