# Лабораторная работа №2 по дисциплине "АОИС"

## Построение СКНФ и СДНФ на основании таблиц истинности

Этот проект содержит программу для анализа булевых функций и выполнения всех требуемых операций лабораторной работы №2.

## Структура проекта

### Основные файлы

| Файл | Описание |
|------|----------|
| `main.py` | Главный файл, анализатор булевых функций |
| `parser.py` | Парсинг логических выражений |
| `truth_table.py` | Построение таблиц истинности |
| `normal_forms.py` | Построение СДНФ и СКНФ |
| `zhegalkin.py` | Полином Жегалкина |
| `post_classes.py` | Классы Поста |
| `minimization.py` | Минимизация ДНФ/КНФ |
| `dummy_variables.py` | Фиктивные переменные |
| `boolean_derivatives.py` | Булевы производные |
| `test_laba2.py` | Тесты |

## Возможности программы

Программа выполняет следующие операции:

1. **Прием логической функции** произвольного формата с операциями `&`, `|`, `!`, `->`, `~` и до 5 переменных (a, b, c, d, e)
2. **Построение таблицы истинности** на основании функции
3. **Построение СДНФ и СКНФ** на основании таблицы истинности
4. **Вывод числовой формы** для СДНФ и СКНФ
5. **Вывод индексной формы функции**
6. **Определение принадлежности** к классам Поста (T0, T1, S, M, L)
7. **Построение полинома Жегалкина**
8. **Поиск фиктивных переменных**
9. **Булева дифференциация** - частные и смешанные производные (от 1 до 4 переменных)
10. **Минимизация расчетным методом** с выводом стадий склеивания для ДНФ и КНФ
11. **Минимизация расчетно-табличным методом** с таблицей покрытия для ДНФ и КНФ
12. **Минимизация картой Карно** с выводом таблицы для ДНФ

## Установка и запуск

### Требования
- Python 3.8 или выше

### Запуск
```bash
python3 main.py
```

### Примеры выражений
```
a&b                    # Конъюнкция
!a|b                   # Дизъюнкция с отрицанием
!(!a->!b)|c           # Импликация
a~b                    # Исключающее ИЛИ
(a|b)&!a&!c           # Сложное выражение (вариант 2)
```
