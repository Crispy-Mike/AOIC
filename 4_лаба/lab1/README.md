# Лабораторная работа №4 (№6)
## Хеш-таблицы с линейным пробингом

### Тематика: Биология (Вариант 4)

### Реализуемые функции:

1. **CRUD операции:**
   - Create (вставка) - `insert(key, data)`
   - Read (поиск) - `search(key)`
   - Update (обновление) - `update(key, new_data)`
   - Delete (удаление) - `delete(key)`

2. **Структура ячейки хеш-таблицы:**
   - ID - ключевое слово
   - C - флажок коллизий
   - U - флажок «занято»
   - T - терминальный флажок
   - L - флажок связи
   - D - флажок вычеркивания
   - P0 - указатель на следующую запись
   - Pi - данные

3. **Обработка коллизий:** Линейный пробинг

### Хеш-функция:
- V = first_char * 33 + second_char (русский алфавит, 33 буквы)
- h(V) = V mod H + B (где H=20, B=0)

### Требования:
- Размер таблицы >= 20 строк
- Количество записей >= 10
- Количество коллизий >= 2
- Количество цепочек >= 3

### Запуск:
```bash
python3 hash_table.py
```

### Интерактивные команды:
- `insert <ключ> <данные>` - вставка
- `search <ключ>` - поиск
- `update <ключ> <новые данные>` - обновление
- `delete <ключ>` - удаление
- `show` - показать таблицу
- `exit` - выход
